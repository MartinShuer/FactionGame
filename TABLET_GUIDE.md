## 📋 在平板上安装

### 方法1: 直接安装APK（推荐）
1. 将打包好的APK传输到iQOO Pad2 Pro
2. 在平板上打开APK文件
3. 允许安装未知来源应用
4. 安装完成

### 方法2: 使用ADB安装
```bash
adb connect <平板IP地址>
adb install factiongame.apk
```

## 🔧 打包为APK

### 使用Buildozer（Linux/WSL）
```bash
cd android_version
buildozer -v android debug
```

APK文件将生成在 `bin/` 目录中。

### 配置说明
`buildozer.spec` 已针对平板优化：
- **屏幕方向**: `portrait` (竖屏) 或 `landscape` (横屏)
- **全屏模式**: 已启用
- **最低Android版本**: API 21 (Android 5.0)
- **目标Android版本**: API 31 (Android 12)

## 🐛 故障排除

### 问题1: 手掌图标不显示
**解决**: 检查Kivy图形库是否正常加载

### 问题2: 字体太小
**解决**: 已优化为72sp/48sp/42sp/36sp，适合高分辨率平板

### 问题3: 视频不全屏
**解决**: 已设置 `size_hint=(1, 1)` 确保全屏显示

### 问题4: 触摸无响应
**解决**: 点击区域为600×600像素，范围足够大
