✅ 为什么用这个方案？
--------------------
- Google Colab 有 Python 版本冲突（无法解决）
- GitHub Actions 成功率 95%+
- 完全免费，无限次构建
- 一次设置，永久使用


⚡ 超级简单的3步
--------------------

【第1步】创建 GitHub 账号（如果没有）
   访问：https://github.com/signup
   （如果已有账号，跳过）


【第2步】创建仓库并上传文件
   1. 访问：https://github.com/new
   2. 仓库名输入：FactionGame
   3. 选择：Public（公开）
   4. 点击：Create repository
   
   5. 在新页面点击：uploading an existing file
   6. 拖拽这7个文件到浏览器：
      □ main.py
      □ buildozer.spec
      □ hand.png
      □ bg.png
      □ rusher.mp4
      □ defender.mp4
      □ .github\workflows\build-apk.yml（重要！）
   
   7. 点击：Commit changes


【第3步】等待自动构建并下载
   1. 点击仓库顶部的"Actions"标签
   2. 看到"Build Android APK"正在运行
   3. 等待30-40分钟（绿色✓表示成功）
   4. 点击构建记录
   5. 下载"FactionGame-APK"


🎉 完成！您得到了 APK！
